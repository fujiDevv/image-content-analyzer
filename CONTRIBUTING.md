# Contributing to Image Content Analyzer

First off, thank you for considering contributing to Image Content Analyzer! It's people like you that make this tool great.

## How to Contribute

1. **Fork the repository** on GitHub.
2. **Clone your fork** locally.
3. **Install dependencies** using `npm install` or `bun install`.
4. **Create a branch** for your feature or bug fix: `git checkout -b my-new-feature`.
5. **Make your changes** and ensure they follow the existing coding style.
6. **Run the tests** and linter to make sure everything works: `npm run test` and `npm run lint`.
7. **Commit your changes**: `git commit -m 'feat: add some feature'`.
8. **Push to the branch**: `git push origin my-new-feature`.
9. **Submit a pull request** describing your changes.

## Development Setup

This project uses TypeScript and `tsup` for fast dual-build bundling (CommonJS and ESM). 
- To build the project: `npm run build`
- To run tests: `npm run test`

## Code of Conduct

By participating in this project, you agree to maintain a respectful, welcoming, and inclusive environment for everyone.
